Task 2 Explanation: 

We compared Python lists vs NumPy arrays in terms of:
Memory
* Python lists store objects → more memory
* NumPy arrays store elements in contiguous memory → efficient
Speed
* Lists require loops
* NumPy uses vectorized operations, making it significantly faster.
